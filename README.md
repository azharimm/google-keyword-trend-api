# Google Keyword Trends API

## 1. Trends
```
[ENDPOINT] /location
```
```
[EXAMPLE] http://api-google-keyword-trends.herokuapp.com/trends
```

## 2. Country Trends
```
[ENDPOINT] /trends/:country
```
```
[EXAMPLE] http://api-google-keyword-trends.herokuapp.com/trends/indonesia
```

## 2. Countries
```
